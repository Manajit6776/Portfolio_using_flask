Sales Data Analysis Project

Objective

Analyze sales data using Pandas to extract key insights and visualize trends.

Key Steps

  • Data Loading - Import CSV file using pd.read_csv()
  • Data Cleaning - Handle missing values, convert date formats, and ensure correct data types
  • Exploratory Analysis - Calculate summary statistics, total sales, and category-wise distribution
  • Aggregation - Use groupby() to analyze sales by month, category, and region
  • Visualization - Generate line charts, bar graphs, and pie charts to display trends
  • Advanced Analysis (Optional) - Correlation heatmap and time-series decomposition

Deliverables

  • Jupyter Notebook with analysis code
  • Cleaned dataset (cleaned_sales_data.csv)
  • Charts (e.g., monthly_sales_trend.png)
  • README.md with project summary

Tools Used

  • Python, Pandas, Matplotlib, Seaborn

Outcome

Identifies top-performing categories, sales trends, and regional performance for data-driven decision-making.
